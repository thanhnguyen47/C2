Web Security Challenges

Server-Side Topics

SQL Injection

Exploit vulnerable SQL queries to extract or modify data.
Authentication

Understand and exploit flaws in login and session management.
Path Traversal

Bypass access controls to read sensitive files on the server.
Command Injection

Inject system commands to execute on the server.
Access Control

Bypass restrictions to access unauthorized functionality or data.
File Upload Vulnerabilities

Bypass filters to upload malicious files like web shells.
Race Conditions

Exploit timing issues in applications to gain unauthorized access.
Server-Side Request Forgery (SSRF)

Exploit server-side applications to make requests on behalf of the server.
XXE Injection

Exploit XML parsers to read arbitrary files or make network requests.
NoSQL Injection

Exploit NoSQL databases by injecting malicious queries.

Client-Side Topics

Cross-Site Scripting (XSS)

Inject malicious scripts into web pages viewed by other users.
Cross-Site Request Forgery (CSRF)

Exploit authenticated users to perform unintended actions.
Cross-Origin Resource Sharing (CORS)

Misconfigurations that allow unauthorized access to resources.
Clickjacking

Trick users into clicking on something different from what they intend.
DOM-Based Vulnerabilities

Manipulate client-side code execution and access user data.

Advanced Topics

Insecure Deserialization

Exploit deserialization vulnerabilities to execute arbitrary code.
Server-Side Template Injection (SSTI)

Inject malicious template code into the server-side rendering engine.
OAuth Authentication

Bypass OAuth authorization to gain unauthorized access.
JWT (JSON Web Tokens)

Exploit weaknesses in JWT handling for unauthorized access.
